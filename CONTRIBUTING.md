## Thanks for helping with pkmapr

Whether you’re fixing a bug, adding a feature, or reporting an issue —
contributions are welcome.

### Found a bug?

Open an issue and include: - A reproducible example (use
reprex::reprex()) - Your sessionInfo() output - What you expected vs
what happened

### Have an idea?

Open an issue with the label “enhancement” and describe what you’d like
to see.

### Want to write code?

1.  Fork the repo
2.  Create a branch: git checkout -b your-feature-name
3.  Make your changes
4.  Run devtools::check()
5.  Push and open a Pull Request

I’ll review within a few days.

### Questions?

Open an issue with the label “question” or just ask.
