# pkmapr

pkmapr provides a tidy interface to Pakistan’s official administrative
boundary data from OCHA/HDX. Download province, district, and tehsil
geometries as sf objects, look up official names and codes, and prepare
spatial data structures for use with spdep, ggplot2, and leaflet.
