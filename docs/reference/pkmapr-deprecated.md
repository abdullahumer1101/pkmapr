# Deprecated functions in pkmapr

These functions were deprecated. Please use the new `pk_*` functions
instead.

## Usage

``` r
pak_crs_suggest(...)

pak_dictionary(...)

pak_join(...)

pak_area(...)

pak_distance(...)

pak_centroid(...)

pak_bbox(...)

pak_project(...)

pak_neighbors(...)

pak_points_in(...)

pak_buffer(...)

pak_union(...)

pak_intersect(...)

pak_map(...)

pak_map_interactive(...)

pak_basemap(...)
```

## Value

No return value, called for side effects (deprecation warnings).
